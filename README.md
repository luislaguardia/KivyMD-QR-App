# KivyMD QR App

## Instructions

To run the KivyMD QR App, please follow the steps below:

1. **Upload Files:**
   - Upload `main.py` and `main.kv` to Google Colab.

2. **Run Cells:**
   - Execute cells 1 to 5 one at a time.
